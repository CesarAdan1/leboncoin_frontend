- [JS Cookie](https://github.com/js-cookie/js-cookie)
- [Axios](https://github.com/axios/axios)

## Running the project

Clone this repository :

```
git clone https://github.com/Cebri63/leboncoin-2003.git
cd leboncoin-2003
```

Install packages :

```
npm install
```

When installation is complete :

```bash
npm start
```
